# 🕹️ LevelUp Games Portal

פורטל משחקים אינטראקטיבי המשלב חוויית משחק מודרנית עם ניהול משתמשים ושיאים. הפרויקט כולל את המשחקים הקלאסיים **סנייק** ו**משחק הזיכרון**.

---

## 📝 אודות הפרויקט
האתר נבנה כפרויקט Web המציג יכולות עבודה עם לוגיקה מורכבת בצד לקוח (JavaScript), ניהול מצבים (State Management) ושמירת נתונים מקומית. הפורטל מציע חוויה תחרותית שבה כל משתמש יכול לראות את השיא שלו לצד "טבלת האלופים" של האתר.

---

## 🚀 תכונות עיקריות (Features)

### 👤 מערכת משתמשים
* **הרשמה והתחברות:** אימות משתמשים ושמירת פרטים ב-`LocalStorage`.
* **דשבורד אישי:** הצגת שלום למשתמש המחובר וגישה מהירה למשחקים.

### 🐍 משחק הסנייק (Snake)
* **אלגוריתם חכם:** מנגנון המונע מהאוכל להופיע על גוף הנחש.
* **ניהול שיאים:** שמירה אוטומטית של הניקוד הגבוה ביותר למשתמש.

### 🧠 משחק הזיכרון (Memory Game)
* **רמות קושי:** קל (12 קלפים), בינוני (18 קלפים), וקשה (24 קלפים).
* **מנגנון טיימר:** מעקב מדויק אחר זמן הסיום ושיפור ביצועים.

### 🏆 טבלת מובילים (Leaderboard)
* **דירוג גלובלי:** הצגת שלושת המקומות הראשונים מכלל המשתמשים במחשב.
* **זיהוי אישי:** הדגשת המיקום של המשתמש הנוכחי בתוך הטבלה (גם אם הוא לא בטופ 3).

## 🛠️ טכנולוגיות ששימשו בפרויקט
* **HTML5:** מבנה האתר ושימוש ב-Data Attributes.
* **CSS3:** עיצוב מתקדם, Flexbox, Grid, ואנימציות למשחקים.
* **JavaScript (Vanilla):** לוגיקת המשחקים, ניהול מערכים, פונקציות מיון ואירועי לחיצה.
* **LocalStorage API:** ניהול בסיס נתונים מקומי למשתמשים ושיאים.
